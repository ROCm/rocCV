Construct a new Op Custom Crop object. The object can be used to crop a region of interest from an image.

Limitations:

Input: Supported TensorLayout(s): [TENSOR_LAYOUT_NHWC] Channels: > 0 && <= 4 Supported DataType(s): [DATA_TYPE_U8]

Output: Supported TensorLayout(s): [TENSOR_LAYOUT_NHWC] Channels: > 0 && <= 4 Supported TensorLayout(s) [DATA_TYPE_U8]
