Executes the Gamma Contrast operation. The operation adjusts the gamma contrast on the images in a tensor. A gamma tensor is given as input to apply a different gamma value on each image in the input tensor.

Limitations:

Input: Supported TensorLayout(s): [NHWC, HWC] Channels: [3] Supported DataType(s): [U8]

Output: Supported TensorLayout(s): [NHWC, HWC] Channels: [3] Supported DataType(s): [U8]

Parameters

[out]outputThe output tensor with gamma correction applied.
[in]gammaOne dimensional tensor with shape N, containing the gamma values for each image in the batch
[in]deviceThe device type to execute on (CPU/GPU).
